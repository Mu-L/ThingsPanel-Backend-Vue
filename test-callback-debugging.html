如果看不到 InteractionManager 日志

说明回调注册或调用出现问题，需要检查：

  • InteractionManager 是否正确初始化
  • registerDataExecutionTrigger 是否成功注册
  • 回调函数绑定是否正确
  • 是否有异步执行导致的上下文丢失

🎯 诊断结果分析

情况1：看到了完整日志链

说明回调链路正常，问题可能在于：

  • HTTP 数据源映射没有找到
  • 配置格式转换有问题
  • VisualEditorBridge 执行失败

情况2：ConfigEventBus 调用完成但 InteractionManager 没响应

说明回调注册或调用有问题：

  • InteractionManager 初始化失败
  • 回调函数绑定错误
  • 异步执行上下文丢失

情况3：ConfigEventBus 就没有调用

说明配置事件分发有问题：

  • 事件格式不正确
  • shouldTriggerExecution 没有设置
  • 事件监听器注册失败
