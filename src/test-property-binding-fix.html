🔍 新增调试信息

重点关注 VisualEditorBridge 的新增调试日志，它会显示：

  • resolvedConfigKeys - 配置对象的所有键
  • hasDataSources / dataSourcesLength - 是否有数据源数组
  • hasRawDataList - 是否有原始数据列表
  • dataSourceKeys - 所有以"dataSource"开头的键
  • finalDataSourcesCount - 最终解析出的数据源数量

🚨 问题排查指南

如果仍然显示 finalDataSourcesCount: 0

说明配置结构与解析逻辑不匹配，请查看 resolvedConfig 样本数据，确认数据源存储格式

如果 hasHttpDataItems: false

说明数据源类型不是 'http' 或结构不正确，检查数据源配置的 type 字段

如果参数绑定错误

查看 DataItemFetcher 的路径参数解析日志，确认是否从 EditorStore 读取到最新值
