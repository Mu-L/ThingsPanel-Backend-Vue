修复 ConfigurationImportExport.ts findIndex undefined 错误

🐛 问题描述

错误信息：

Cannot read properties of undefined (reading 'findIndex')
at ConfigurationImportExport.ts:1153:56
at SingleDataSourceImporter.importSingleDataSource

问题原因：

  • 在调用 existingConfig.dataSources.findIndex() 时
  • existingConfig.dataSources 可能是 undefined 或 null
  • 可能与之前的 HTTP 表单参数修改相关，影响了数据结构

✅ 修复方案

在 ConfigurationImportExport.ts:1152-1156 添加了安全检查：

// 🔧 确保 dataSources 数组存在 if (!existingConfig.dataSources || !Array.isArray(existingConfig.dataSources)) { existingConfig.dataSources = [] console.warn(`⚠️ [SingleDataSourceImporter] dataSources 数组不存在或不是数组，已重置为空数组`) }

修复内容：

  • ✅ 检查 dataSources 是否存在
  • ✅ 检查 dataSources 是否是数组类型
  • ✅ 如果不存在或类型错误，重置为空数组
  • ✅ 添加警告日志便于调试
